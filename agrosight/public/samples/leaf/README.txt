Leaf Health demo images (PlantVillage-style)
============================================
Upload these on /inspect → Leaf Health mode.

Folders:
  tomato/  maize/  apple/

Each folder has:
  healthy_*           → expect Healthy
  surface_defect_*    → expect Surface Defect
  blight_mold_*       → expect Blight / Mold
