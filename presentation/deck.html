Price connect

How fair price is built

1 · Produce Inspect Fresh / Borderline / Rotten → Grade A · B · C + defect %
2 · Fetch mandi modal data.gov.in Agmarknet API (free) · state + commodity · snapshot fallback if offline
3 · Apply grade rule A = modal × 1.10 · B = modal · C = modal × 0.80
4 · Use everywhere Prices screen · Market list price · buyer “fair price” badge

Example: Modal ₹1,733 → Grade A ≈ ₹1,906 · Grade C ≈ ₹1,386
